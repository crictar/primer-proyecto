Comandos para Git

Antes del primer commit

git config -l .- Muestra una lista de las configuraciones de git, como el nombre de usuario o email
git config --global user.name "John Doe" .- Configura el nombre de usuario a John Doe
git config --global user.email abc@ejemplo.com .- Configura el email del usuario a abc@ejemplo.com (importante para github)

Crear commit

git status .- Indica tu posicion actual, commits, archivos trackeados y cambios realizados
git add . .- Lleva los archivos a staging area para hacerles seguimiento o trackearlos
git rm . .- Quita el seguimiento a los archivos
git commit -m "Mensaje" .- Crea un commit con un mensaje de descripcion

Analizar los cambios

git show archivo.txt .- Compara "archivo.txt" con sus versiones anteriores
git log .- Muestra todos los commits realizados con sus argumentos (ej. b90d120d8e40d4)
git log --stat .- Muestra todos los commits realizados con sus argumentos y todos los cambios realizados
git log archivo.txt .- Muestra los commits en los que esta uncluido "archivo.txt"
git diff .- Muestra las diferencias entre el commit actual y los anteriores
q .- Salir cuando aparezca (END)

Regresar a una version anterior

git log .- Muestra todos los commits realizados con sus argumentos (ej. b90d120d8e40d4)
git checkout b90d120d8e40d4 .- Lleva al directorio de trabajo el commit con el argumento b90d120d8e40d4
git checkout b90d120d8e40d4 archivo.txt .- Lleva al directorio de trabajo el archivo.txt del commit b90d120d8e40d4

Crear y pasar entre ramas

git branch .- Muestra todas las ramas, y marca en que rama estas con un *
git branch ejemplo .- Crea una nueva rama llamada ejemplo
git checkout ejemplo .- Pasa a la rama ejemplo
git checkout -b ejemplo .- Realiza los dos comandos anteriores en uno solo

Merge/ unir ramas

git checkout main .- Vuelve a la rama main
git merge ejemplo .- Fusiona los ultimos commits de las ramas main y ejemplo

Bajar archivos del repositorio remoto (github)

git fetch origin main .- Baja los archivos del repositorio remoto al local
git pull origin main .- Baja los archivos del repositorio remoto y lo sube al directorio de trabajo

Subir archivos al repositorio remoto (github)

git remote add origin https://url .- Crea un vinculo con el repositorio remoto (https://url) llamado origin
git remote .- Muestra la coneccion con el repositorio remoto
git push origin main .- Sube la rama main al repositorio remoto
