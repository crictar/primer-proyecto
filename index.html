Comandos para Git

Antes del primer commit

git config -l .- Muestra una lista de las configuraciones de git, como el nombre de usuario o email
git config --global user.name "John Doe" .- Configura el nombre de usuario a John Doe
git config --global user.email abc@ejemplo.com .- Configura el email del usuario a abc@ejemplo.com (importante para github)

Crear commit

git status .- Indica tu posicion actual, commits, archivos trackeados y cambios realizados
git add . .- Lleva los archivos a staging area para hacerles seguimiento o trackearlos
git rm . .- Quita el seguimiento a los archivos
git commit -m "Mensaje" .- Crea un commit con un mensaje de descripcion

Analizar los cambios

git show archivo.txt .- Compara "archivo.txt" con sus versiones anteriores
git log archivo.txt .- Muestra los commits en los que esta uncluido "archivo.txt"
git diff .- Muestra las diferencias entre el commit actual y los anteriores

Crear y pasar entre ramas

git branch .- Muestra todas las ramas, y marca en que rama estas con un *
git branch ejemplo .- Crea una nueva rama llamada ejemplo
git checkout ejemplo .- Pasa a la rama ejemplo
